# Changelog

## [2.0.0] - 2025-04-25

### Added
- Complete fullstack platform with React 19 frontend and tRPC/Hono backend
- Model Explorer with 11 pre-seeded open-source AI models
- Universal Chat interface supporting multi-model conversations
- Agent Builder with memory, tools, and knowledge base attachments
- Knowledge Base / RAG system supporting ChromaDB, Qdrant, Weaviate, Milvus, pgvector
- Workflow Engine with node-based pipeline designer
- Integration Manager for Ollama, OpenAI, Anthropic, Google, HuggingFace
- MCP Server support for Model Context Protocol
- OAuth 2.0 authentication with role-based access control
- MySQL database with Drizzle ORM and 12 relational tables
- Docker and Docker Compose configuration
- CI/CD pipeline with GitHub Actions
- Comprehensive documentation and architecture guides
