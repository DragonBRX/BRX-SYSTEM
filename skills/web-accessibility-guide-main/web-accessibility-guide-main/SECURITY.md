# Security

## Reporting a vulnerability

If you believe you have found a security vulnerability in this project, please report it **privately** so we can address it before wider disclosure.

**Preferred options (use whichever applies):**

1. **[SECURITY_CONTACT]** — Replace with a maintainer email or other private channel you monitor.
2. **GitHub** — Use [GitHub Security Advisories](https://docs.github.com/code-security/security-advisories) for this repository if enabled.

Please include enough detail to reproduce or understand the issue (affected component, steps, impact). Do not open a public issue for undisclosed vulnerabilities.

## What to expect

Maintainers will acknowledge receipt when they can and will work toward a fix or mitigation. We do not guarantee specific response or resolution timelines.

Thank you for helping keep users and contributors safe.
